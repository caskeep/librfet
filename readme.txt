libmfet
-------

Library for parsing and evaluating the MFET
(Math Function Expression Text) language.

To compile with GCC:
first get the folders:
    * ../algorithms
    * ../lib_std
then execute: make

"main.c" enables running the MFET Calculator on
the command line interface. To compile it, use:
    make -f make_main
To run it, use: ./mfet
