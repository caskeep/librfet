librfet
-------

Library for parsing and evaluating the
Rhyscitlema Function Expression Text
(RFET) language.

To compile with GCC,
first get the folders:
    * ../algorithms
    * ../lib_std
then execute: make

"main.c" enables running the RFET Calculator on
the command line interface. To compile it, use:
    make -f make_main
To run it, use: ./rfet
